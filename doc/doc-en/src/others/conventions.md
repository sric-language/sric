## Coding Conventions

### Source Files
Source files must use UTF-8 encoding.

### Indentation
Use 4 spaces for indentation:
```sric
    if (cond) {
        doTrue
    }
    else {
        doFalse
    }
```
### Naming
- Type names use PascalCase
- Module names, functions and variables use camelCase
- Never use ALL_CAPS naming style
